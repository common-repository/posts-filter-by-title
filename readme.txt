== Description ==

This is a simple plugin which adds another drop-down box of Post Titles into the filter section on posts listing page in the wordpress backend. This is usefull when we have more then 99 posts in the site and want to choose a particular post without having to remember and search by post title or navigate through the pagination.

== Installation ==

*   Download and unpack the archive containing the plugin to the /wp-content/plugins/ directory

*   Activate the plugin through the \'Plugins\' menu in WordPress. Thats it.

*   Go to posts listing page in the backend and use the posts title filter.

== Frequently Asked Questions ==

Q: Can I use this for custom post types filter ?
